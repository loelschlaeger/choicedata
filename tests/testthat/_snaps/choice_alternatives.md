# choice_alternatives can be printed

    Code
      choice_alternatives(J = 3, alternatives = c("la", "le", "lu"))
    Message
      
      -- Choice alternatives  
    Output
      * la*
      * le
      * lu

---

    Code
      choice_alternatives(J = 4, ordered = TRUE)
    Message
      
      -- Choice alternatives (ordered) 
    Output
      * A
      * B
      * C
      * D
